mcpelauncher-installer - An Installer For MCPELauncher

Supproted distros:
Ubuntu and Ubuntu based distros (16.04,18.04,18.10,19.04)

                                                                How to Install:
Ubuntu 16.04:
$ git clone https://github.com/glowiak/mcpelauncher-installer.git && cd mcpelauncher-installer && sh xenial.sh
Ubuntu 18.04 and 18.10:
$ git clone https://github.com/glowiak/mcpelauncher-installer.git && cd mcpelauncher-installer && sh bionic.sh
Ubuntu 19.04:
$ git clone https://github.com/glowiak/mcpelauncher-installer.git && cd mcpelauncher-installer && sh disco.sh
MacOS: (I not created installer for Mac, but is orginal dmg image)

MCPELauncher website: http://mcpelauncher.readthedocs.io/en/latest
